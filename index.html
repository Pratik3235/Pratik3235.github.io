Coursera Clone

The Coursera Clone project is a frontend UI replica of the popular online learning platform, Coursera. It provides an organized layout to display various online courses, degree programs, and certificates. The project features Firebase integration to manage course data and user interactions, allowing dynamic content retrieval. Users can explore course listings, degree details, and community features seamlessly, while also being able to access personalized content after login.

Tech Stack: HTML, CSS, JavaScript, Node.js, Firebase, API

CraftBridge - Traditional Craft Sharing Platform

CraftBridge is a full-stack web application designed to help users share, explore, and preserve traditional crafts. The platform allows artisans to upload their work and enables visitors to browse, search, and view detailed information on various crafts. Key features include user authentication, the ability to upload and manage crafts, a paginated explorer for browsing, and a personalized user dashboard for managing uploads. Users can also edit their craft entries and create a profile page.

Tech Stack: React.js, React Router, CSS Modules, Custom Styles, Express.js, Node.js, MongoDB, Mongoose, Axios, dotenv, cors, body-parser, nodemon
